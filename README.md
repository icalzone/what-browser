# SAAS debug info

SAAS debug information page, inspired by whatbrowser.org (now dead) and dropbox-debug.com.

## Getting started

__Requirements__

- NodeJS > 8
- npm

Download & install dependencies

```
git clone https://github.com/buren/saas-debug
cd saas-debug
npm install
```

Build
```
npm run build
# creates dist/bundle.js
```
